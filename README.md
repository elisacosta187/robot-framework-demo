## Useful commands

| Command                                           | Description                                                   |
| ------------------------------------------------- | ------------------------------------------------------------- |
| `robot --output=./test-results ./tests`           | Run all tests.                                                |
| `robot --output=./test-results -debug ./tests`    | Run all tests in debug mode.                                  |
